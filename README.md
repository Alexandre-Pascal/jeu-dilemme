# Le Dilemme Parfait

Jeu social en temps réel — dépôt initialisé. Le code du monorepo (React, Fastify, Socket.io) sera ajouté ensuite.

## Prérequis

- Node.js (LTS)
- [pnpm](https://pnpm.io/)
- Docker (optionnel, pour PostgreSQL en local)

## Dépôt Git

```bash
git clone https://github.com/<ton-compte>/jeu-dilemme.git
cd jeu-dilemme
```

Pour lier ce dossier à un dépôt GitHub vide après création sur github.com :

```bash
git remote add origin https://github.com/<ton-compte>/jeu-dilemme.git
git push -u origin main
```
